7+ Taskbar Tweaking Library is a library for manipulating the taskbar of Windows 7 and above.
Currently, the following operating systems are supported:
Windows 7, Windows 8, Windows 8.1, and Windows 10.

The library is free for non-commercial use. For commercial use, please see the link below.

For more information, please refer to the library web page:
